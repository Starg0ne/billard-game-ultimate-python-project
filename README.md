A LIRE

Fonctionne sur python3 .

Pour lancer le jeu, assurez-vous d'avoir installés les prérequis dans requirement.txt .

Pour le bouton "NE PAS APPUYER", appuyez puis entrez "C0DE Administrateur" pour ouvrir la page spéciale. Vous pouvez regler les parametres comme bon vous semble, ou utiliser l'option "preset" pour obtenir une version customisée, tout en gardant des valeurs cohérentes pour garder un jeu jouable.
